📖 使用说明

基于后端代理 + html2canvas 引擎，可靠获取任意网页

1

启动后端服务器

本地运行: node server.js · 部署到 Vercel 自动生效

2

输入网址配置参数

填入目标网址，选择格式、截图宽度等参数

3

一键截图

点击开始截图，自动渲染并导出下载

🚀 部署到 Vercel（推荐，免费，无需管理服务器）

# 1. 推送到 GitHub
git push
# 2. 在 vercel.com 导入仓库，自动部署
# 3. 部署完成，直接可用 ✅

Vercel 会自动运行 api/fetch.js 做后端代理，无需自己搭服务器。

💻 本地开发（需要 Node.js）

# 安装依赖
npm install
# 启动
node server.js

本地访问 http://localhost:3000

🐍 Python 备用方案

# 安装依赖
pip install flask requests
# 启动
python server.py
